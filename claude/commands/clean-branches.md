# ローカルの不要ブランチを削除

ローカルの不要ブランチを安全に削除します。

## 手順

1. `git fetch --prune` でリモートの最新状態を取得しつつ、削除済みリモートブランチのトラッキング情報を削除
2. 削除候補ブランチを以下の条件で洗い出してユーザーに一覧表示する
   - **マージ済みブランチ**: `git branch --merged main`（またはデフォルトブランチ）で検出、ただし `main`, `master`, `develop` は除外
   - **リモート追跡なしブランチ**: リモートに対応するブランチが存在しない（または削除済み）ローカルブランチ
3. ユーザーに削除するブランチを確認
   - `$ARGUMENTS` が指定された場合はそのブランチ名のみ対象とする
   - 指定がない場合は全削除候補を表示して確認
4. ユーザーが承認したブランチのみ `git branch -d`（マージ済み）または `git branch -D`（未マージ）で削除
5. 削除結果を報告

## 注意

- 現在チェックアウト中のブランチは削除しない
- `main`, `master`, `develop` ブランチは絶対に削除しない
- 未マージブランチを `-D` で強制削除する場合は必ずユーザーに警告して個別に確認する
- 削除前に必ずブランチ一覧をユーザーに提示し、承認を得てから実行する
